HTML5 Canvas utility to render texts in retro style and more.

Get Started

You only need to:

  • include the pxxl-0.1.min.js script.
  • include JQuery (used only for ajax requests. this dependency should be removed soon)

To get started immediately, create a canvas element and just call the 'PixelText' function: 

  
      PixelText($("canvas") , { 
        text: 'hello world!',
        font : 'fonts/c64.bdf',
        style : 'squares',
        scale: '10',

        fill: "#66f",
        stroke: "black"
      });
